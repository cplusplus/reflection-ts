\rSec0[dcl.decl]{Declarators}

\rSec1[dcl.name]{Type names}

\pnum
In C++ [dcl.name], apply the following changes:

\begin{std.txt}
To specify type conversions explicitly, and as an argument of \tcode{sizeof},
\tcode{alignof}, \tcode{new}, \removed{or} \tcode{typeid}, \added{or
\tcode{reflexpr},} the name of a type shall be specified.
\end{std.txt}
