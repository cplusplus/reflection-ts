\rSec2[diff.lex]{Clause \ref{lex}: lexical conventions}

In C++ [diff.lex], modify the first paragraph:

\begin{std.txt}
\diffref{lex.key}
\change New keywords.
\rationale Required for new features.
The \tcode{requires} keyword is added
to introduce constraints through a \grammarterm{requires-clause} or
a \grammarterm{requires-expression}. The \tcode{concept} keyword is
added to enable the definition of concepts (\cxxref{temp.concept}).
\added{The \tcode{reflexpr} keyword is added to introduce meta-data through a \grammarterm{reflexpr-specifier}.}
\effect
Valid ISO \CppXVII{} code using \tcode{concept}\removed{ or}\added{,} \tcode{requires}\added{, or \tcode{reflexpr}}
as an identifier is not valid in this International Standard.
\end{std.txt}
